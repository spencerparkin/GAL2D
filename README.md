# GAL2D

This is a simple graphics abstraction layer for 2D graphics.
The idea is that this could serve as the basis for something like a
simple cross-platform GUI framework.

Note that GAL2D makes use of the following dependencies...
 * [FreeType](https://freetype.org/)
 * [STB](https://github.com/nothings/stb)
 * [OpenGL](https://www.opengl.org/)
 * DirectX11
 * DirectX12
